# Skills Overview

## Purpose

This is the skill index — read it **first** whenever a practitioner describes a task, to identify the single most relevant skill before loading anything else. Match the task to one skill, then load only that skill file in full, which keeps the context window efficient rather than pulling every skill into memory at once.

## How to use it

- Read this index first when the practitioner describes a task.
- Match the task description to the most relevant skill below.
- Load only the matched skill file in full before proceeding.
- If no skill matches, use the knowledge files directly and proceed without a skill.

## Skill index

- **Account Audit** — `skills/account-audit.md` — Structured diagnostic across five dimensions: conversion tracking, structure, bidding, creative, search term quality. *Reach for it when* onboarding a new client, running a health check, or diagnosing a performance drop.
- **Campaign Review** — `skills/campaign-review.md` — Fast weekly or fortnightly campaign performance review. *Reach for it when* checking in on active campaigns, preparing for a client call, or after a budget or target change.
- **Monthly Report Brief** — `skills/monthly-report-brief.md` — Generates the written narrative for a monthly client report. *Reach for it when* producing monthly reporting, preparing for a performance review, or writing a client summary.
- **Search Term Analysis** — `skills/search-term-analysis.md` — Analyses search term reports with conversion lag offsets, flags waste, promotes converting terms, and outputs importable negative keyword lists. *Reach for it when* running weekly search term hygiene, after launching broad match campaigns, or when CPA has risen unexpectedly.
