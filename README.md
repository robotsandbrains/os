# Robots & Brains OS

## What this is
One-line pitch: an open-source AI operating system for paid media practitioners — best practices, platform knowledge, and runnable skills, all in markdown.

## Who it's for
The target practitioner (in-house, agency, freelance paid media) and the problem it solves.

## The core idea
How "robots" (Claude + skills) and "brains" (the practitioner's judgment) combine; principle-led, human-in-the-loop.

## What's inside
- **Best practices** — durable principles for Google Ads (and Meta, coming).
- **Campaign types** — platform mechanics per campaign format.
- **Contexts** — how strategy shifts by business model (lead-gen, ecomm).
- **Skills** — runnable workflows (audit, review, reporting).
- **Obsidian template** — vault structure for managing client accounts.
